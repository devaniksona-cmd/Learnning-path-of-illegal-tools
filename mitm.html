CATEGORY 07

MITM / Network
Interception

ARP poisoning, packet sniffing, and network interception tools. Violates wiretapping laws in every jurisdiction. Federal felony to intercept others' traffic — even on open WiFi.

ettercap Sends forged ARP reply packets to both the victim and the router, poisoning their ARP caches so both devices believe your MAC address is the other party. All traffic between victim and internet now flows through your machine first. You can read all unencrypted traffic, inject custom content into HTTP pages, strip SSL/TLS with SSLstrip to downgrade HTTPS connections, and capture login credentials in plaintext. ARP poisoning + traffic interception on networks you don't own = wiretapping felony under ECPA. sudo apt install ettercap-graphical sudo ettercap -T -M arp /victim// /gateway// ⚠ Installs — MITM on others = wiretapping
bettercap Modern all-in-one network attack framework. Beyond basic ARP spoofing, it includes DNS spoofing (redirect victim's DNS queries to your machine), HTTP/S sniffing with automatic SSL certificate spoofing, JavaScript injection into HTTP responses to deliver custom payloads, BLE (Bluetooth Low Energy) scanning and impersonation, and a real-time interactive web UI for managing all active attacks simultaneously. Full network attack framework — ARP + DNS + SSL stripping + JS injection. Criminal on any unauthorized network. sudo apt install bettercap sudo bettercap -iface eth0 -eval 'arp.spoof on' ⚠ Installs — unauthorized MITM = criminal
arpspoof Sends a continuous stream of forged ARP replies to the victim machine, telling it that the attacker's MAC address is the gateway's IP. Simultaneously sends forged ARPs to the gateway claiming to be the victim. With IP forwarding enabled on the attacker's machine, all traffic between victim and internet passes through the attacker for interception, modification, or logging. ARP cache poisoning redirects traffic through your machine. Intercepting that traffic = federal wiretapping offense. sudo apt install dsniff sudo arpspoof -i eth0 -t victim_ip gateway_ip ⚠ Installs — illegal on others' networks
Wireshark (on others) Puts your network interface into promiscuous mode to capture every packet that reaches your NIC — including packets not addressed to you. On a hub or after ARP poisoning, this includes other users' traffic. Wireshark decodes hundreds of protocols in real time, showing you usernames, passwords, session cookies, emails, and file transfers from everyone on the same network segment in readable form. Capturing packets not addressed to you = unauthorized interception = wiretapping felony under ECPA. sudo apt install wireshark wireshark -i eth0 -k # promiscuous mode ⚠ Installs — capturing others = felony
tcpdump (unauthorized) CLI-based packet capture tool that writes raw network packets to a .pcap file. With -i any it captures traffic on all interfaces simultaneously. Can filter by host, port, protocol, or packet content. When run in promiscuous mode on a shared network or after ARP poisoning, captures and logs all traffic passing through your interface including other users' unencrypted credentials and session tokens. Logging others' network traffic without authorization = electronic surveillance = criminal offense. sudo apt install tcpdump sudo tcpdump -i eth0 -w capture.pcap ⚠ Installs — unauthorized sniffing = illegal
mitmproxy (on others) Runs a transparent HTTP/S proxy that intercepts and decrypts TLS traffic by presenting its own certificate to clients. When configured as a transparent proxy (combined with iptables rules to redirect traffic), all HTTP/S traffic from the victim is decrypted, logged in full with request/response bodies, and re-encrypted before forwarding. You see every API call, form submission, and cookie in plaintext. Decrypting and logging others' TLS-protected communications without consent = criminal wiretapping. pip3 install mitmproxy mitmproxy --mode transparent ⚠ Installs — legal only on your own traffic
⚠ The Electronic Communications Privacy Act (ECPA) in the USA and equivalent laws globally make intercepting others' network traffic a federal felony. This applies even on an open, unencrypted WiFi network. "The network was open" is not a legal defence.
