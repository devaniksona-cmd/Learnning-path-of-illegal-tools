CATEGORY 01

DoS / DDoS
Tools

Tools that flood servers with traffic to cause denial of service. Always excluded from every bug bounty scope. Criminal under CFAA and equivalent laws worldwide.

TOOL WHAT IT DOES WHEN USED WHY ILLEGAL INSTALL COMMAND RUN COMMAND OUTCOME
LOIC Floods the target IP and port with tens of thousands of TCP, UDP, or HTTP packets per second from your machine. The server's CPU and network interface become saturated — legitimate users get connection refused or timeout errors. Visible immediately in server logs. Violates CFAA §1030 — intentional damage to protected computers. DoS is never in scope. sudo apt install loic loic /target:192.168.1.1 /port:80 ❌ Removed from repos
HOIC Uses booster scripts to randomize headers and URLs, amplifying attack volume 3×–1000× beyond LOIC. Sends massive HTTP floods that bypass simple IP-based blocks. Can be coordinated across multiple systems for a true distributed denial-of-service. Amplified DDoS — more severe criminal charge than LOIC. Causes widespread collateral damage. git clone github.com/HOIC python hoic.py -t http://target.com ❌ GitHub took repo down
Slowloris Opens hundreds of partial HTTP connections and sends incomplete headers every few seconds, keeping each connection alive without completing it. Apache and other servers have a fixed thread pool — once all threads are occupied holding Slowloris connections, the server becomes completely unresponsive to real users. Crashes web servers with minimal bandwidth. Real users lose access — criminal disruption. pip3 install slowloris slowloris target.com -p 80 ⚠ Installs — ISP/WAF blocks
GoldenEye Sends HTTP keep-alive requests with random headers, user agents, and cache-control directives to force the server to hold connections and allocate memory per request. Bypasses basic WAF rules by appearing as legitimate browsing traffic until connection count saturates the server. Designed to evade WAFs while still causing complete denial of service. Criminal offense. git clone github.com/jseidl/GoldenEye python goldeneye.py http://target.com ⚠ Installs — WAF blocks
torshammer Routes a Slowloris-style attack through the Tor network, masking your real IP address. Each Tor exit node sends partial HTTP connections to the target. Harder to block because attack traffic appears to come from many different IP ranges around the world. Tor does not provide legal protection — attack origin is still traceable. Criminal offense. git clone github.com/dotfighter/torshammer python torshammer.py -t target.com -p 80 ⚠ Installs — still traceable
hping3 Crafts raw TCP/UDP/ICMP packets at maximum NIC speed using the --flood flag. Can forge source IP addresses (spoofing), target specific ports, and saturate the victim's uplink bandwidth completely. Used to test firewall rules, but here abused to overwhelm network stacks and crash routers. Sends crafted packets at line speed — network device crashes. Severe CFAA violation. sudo apt install hping3 hping3 -S --flood -V -p 80 target.com ⚠ Installs — ISP detects flood
⚠ DoS/DDoS is ALWAYS excluded from bug bounty scope. Running any of these tools triggers ISP blocks, IP bans, and criminal charges. Even brief use on a target is enough for law enforcement to subpoena your ISP.
