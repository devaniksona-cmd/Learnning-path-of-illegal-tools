CATEGORY 03

WiFi Attack
Tools

Wireless password cracking and WPS exploitation tools. Never in scope for any web bug bounty program. A completely separate and illegal domain.

ℹ BLUE = Install Command   PURPLE = Run Command    Click any command to copy.
TOOL WHAT IT DOES WHEN USED WHY ILLEGAL INSTALL COMMAND RUN COMMAND OUTCOME
aircrack-ng Puts your wireless NIC into monitor mode to passively capture WPA2 handshake packets from nearby networks. When a client connects, it sniffs the 4-way handshake. Then runs an offline dictionary or brute-force attack against the captured handshake using your wordlist to recover the WiFi password in plaintext. Cracking WiFi passwords without authorization violates the CFAA and equivalent laws globally. sudo apt install aircrack-ng aircrack-ng -w wordlist.txt capture.cap ⚠ Installs — cracking WiFi = illegal
reaver Exploits a design flaw in the WPS (WiFi Protected Setup) protocol. WPS PINs are validated in two halves — reaver brute-forces each half independently, reducing the search space from 100 million combinations to just 11,000. Can recover a router's WPA2 password in 4–10 hours without needing to crack the password directly. WPS brute-force is a targeted attack on a specific device. Criminal under computer misuse laws. sudo apt install reaver reaver -i wlan0mon -b AA:BB:CC:DD:EE:FF -vv ⚠ Installs — WPS attacks = criminal
wifite Automates the entire WiFi attack workflow — scans nearby networks, selects targets, puts NIC in monitor mode, captures WPA handshakes, deauthenticates clients to force reconnection and handshake capture, then hands off captured files to aircrack-ng or hashcat for offline cracking. Runs everything in sequence with no manual steps. Automated WiFi attack wrapper — fully criminal, no valid bug bounty target ever. sudo apt install wifite sudo wifite --wpa --dict wordlist.txt ⚠ Installs — no BB application
cowpatty Performs offline dictionary attacks against captured WPA/WPA2-PSK handshake files. Can also use pre-computed PBKDF2 hash tables (rainbow tables) specific to a target SSID, dramatically speeding up cracking compared to real-time computation. Often used after aircrack-ng for alternative wordlist or table-based cracking. Offline WPA cracking against captured handshakes from others' networks. Illegal everywhere. sudo apt install cowpatty cowpatty -r capture.pcap -f dict.txt -s SSID ⚠ Installs — cracking = illegal
fern-wifi-cracker GUI-based front-end that wraps aircrack-ng, reaver, and other wireless tools into a point-and-click interface. Scans nearby networks, displays them visually, lets you click a network to attack it, then automatically runs the appropriate attack sequence. Designed to make WiFi hacking accessible to beginners with no CLI knowledge. GUI makes WiFi hacking trivially easy — same criminal liability as CLI tools. sudo apt install fern-wifi-cracker sudo fern-wifi-cracker # launches GUI ⚠ Installs — zero BB use case
kismet Passively monitors all wireless traffic in range across all channels without associating with any network. Logs every beacon frame, probe request, data packet, and hidden SSID it detects. Builds a complete map of nearby devices, their MAC addresses, signal strength, and communication patterns over time. Passive sniffing of traffic from networks you don't own violates wiretapping laws. sudo apt install kismet sudo kismet -c wlan0 ⚠ Installs — sniffing others = illegal
⚠ WiFi hacking requires a physical wireless NIC in monitor mode and is NEVER in scope for web bug bounty programs. No platform will pay you for cracking a WiFi password. This is a completely separate (and illegal) domain.
