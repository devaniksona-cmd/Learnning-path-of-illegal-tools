evilginx2 Acts as a reverse proxy sitting between the victim and the real website. The victim sees a pixel-perfect clone of the real login page at a lookalike domain. When they log in, evilginx intercepts their session cookie in real time — bypassing 2FA entirely because it captures the post-authentication token, not just credentials. Steals authenticated sessions, bypasses MFA. Criminal phishing + identity theft charges. go install github.com/kgretzky/evilginx/v3@latest sudo ./evilginx -p ./phishlets ⚠ Installs — hosting one = criminal
gophish Provides a full phishing campaign management dashboard. You create email templates, upload employee lists, launch bulk phishing emails, and track who clicked your link, who entered credentials, and from which IP. Gives real-time statistics on how many users were compromised by your campaign. Sends phishing emails to real employees. Even with written scope, targets must be clearly defined. go install github.com/gophish/gophish@latest ./gophish # visit :3333 admin panel ⚠ Installs — needs written auth
modlishka Operates as a real-time HTTP reverse proxy that rewrites all URLs and forms on a target website to route through your server. Victims interact with the real site through your proxy — you log all keystrokes, credentials, and 2FA codes as they type them without the victim ever realising they're being intercepted. Real-time keylogging through a proxy. Criminal wiretapping + credential theft. go install github.com/drk1wi/Modlishka@latest modlishka -phishing target.com ⚠ Installs — deploying = illegal
king-phisher Hosts cloned phishing pages and sends templated emails that embed invisible 1×1 pixel tracking images. When a victim opens your email, their IP, browser, and OS are logged. When they click and submit credentials, the data is stored in a local database with timestamps and geolocation data. Email phishing + data harvesting infrastructure. Criminal under anti-phishing statutes. sudo apt install king-phisher king-phisher-server -f config.yml ❌ Removed from default repos
SET (Social Engineer Toolkit) Menu-driven toolkit that automates credential harvesting attacks, spear-phishing emails, infectious media, web jacking, and QR code attacks. Can clone any website in seconds and start a listener. Also generates malicious payloads disguised as PDF files or Office documents for attachment-based attacks. Full attack automation across email, web, and USB vectors. Criminal offense without authorization. sudo apt install set sudo setoolkit ⚠ Kali-only — unauthorized = illegal
HiddenEye Clones popular websites (Facebook, Google, Instagram, PayPal, Netflix) into convincing phishing pages with one command. Hosts them on a local server and generates a URL via ngrok tunnel. When victims enter credentials, HiddenEye logs IP, browser info, credentials, and sends real-time notifications to the attacker. Automated credential theft disguised as popular services. Federal phishing crime. git clone github.com/DarkSecDevelopers/HiddenEye python3 HiddenEye.py ❌ Repo taken down — credential theft
⚠ Phishing is out of scope on 99% of programs. Running a campaign without written permission listing specific targets = lawsuit + criminal charges. Even an employee who clicks your link can trigger legal action against you.
